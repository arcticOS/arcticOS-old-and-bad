## About
arcticOS is an open-source, Linux-based mobile operating system designed for feature phones with E-ink displays.

## Features
- Emulator
- Calculator
- Notes (buggy)

## Planned Features
- Calls and SMS with the SIM7600* series of modems.
- IRC and Matrix messages
- Music Player
- Calendar
- Clock with alarms
- E-mail client
- OTA updates?
